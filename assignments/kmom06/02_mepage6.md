Krav
-----------------------

1.

1. Lägg till din redovisningstext i `redovisning.html`.



### Validering

Se till så koden validerar:

```console
$ cd me
# js
$ npm run linter redovisa/

# html/css
$ dbwebb validate redovisa
```

### Publicering

Glöm inte att publicera koden:

```console
$ cd me
$ dbwebb publish redovisa
```
